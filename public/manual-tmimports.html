Manual do Administrador Global

TM Imports - Sistema ERP Multi-Empresa

1. Dashboard Global

O Dashboard Global apresenta metricas consolidadas de toda a rede:

  • Total de Vendas: Soma de vendas de todas as franquias
  • Total de Franquias: Quantidade de lojas na rede
  • Faturamento: Receita total do periodo
  • Solicitacoes Pendentes: Pedidos de produtos aguardando aprovacao
  • Grafico de Vendas: Evolucao mensal (ultimos 6 meses)
  • Fluxo de Caixa: Entradas vs Saidas com projecao de saldo

2. Gestao de Franquias

Para cadastrar uma nova franquia:

  1. Acesse "Franquias" no menu
  2. Clique em "Nova Franquia"
  3. Preencha os dados da empresa (CNPJ, Razao Social, etc.)
  4. Preencha os dados da loja principal (endereco, telefone)
  5. Crie o usuario administrador da franquia
  6. Clique em "Criar Franquia"
O sistema cria automaticamente: empresa + loja + usuario admin em uma unica operacao.

3. Produtos / Servicos

Cadastro Manual

  1. Acesse "Produtos / Servicos"
  2. Clique em "Novo Produto"
  3. Preencha os dados e salve

Importacao por Planilha

  1. Na mesma tela, clique em "Importar Planilha"
  2. Selecione o arquivo Excel ou CSV
  3. O sistema importa os produtos e classifica automaticamente pela coluna "Categoria do produto"
Colunas esperadas: Codigo, Produto, Preco, Categoria do produto (Moto, Servico, Peca)

4. Estoque Central

O estoque central e o inventario da TM Imports (matriz). Todas as franquias solicitam produtos deste estoque.

Para ajustar o estoque:

  1. Acesse "Estoque Central"
  2. Localize o produto
  3. Clique em "Ajustar" para alterar a quantidade

5. Solicitacoes de Compra

As franquias enviam solicitacoes de produtos para a matriz. Para processar uma solicitacao:

  1. Acesse "Solicitacoes" no menu
  2. Visualize os pedidos pendentes
  3. Para cada solicitacao, voce pode:
    • APROVAR: Aceitar o pedido
    • REJEITAR: Negar o pedido
    • ENVIAR: Apos aprovar, enviar os produtos
Ao clicar em "Enviar", o sistema automaticamente baixa os produtos do estoque central e adiciona no estoque da loja franqueada.

6. Usuarios

Perfis de usuario disponiveis:

  • ADMIN_GLOBAL: Acesso total ao sistema (voce)
  • GESTOR_FRANQUIA: Administra uma loja especifica
  • OPERACIONAL: Acesso limitado (vendas, OS, clientes)

7. Financeiro Consolidado

Contas a Receber

  • Lista todas as contas de todas as lojas
  • Filtro por status (Pendente, Parcial, Pago)
  • Destaque para contas vencidas
  • Baixa com suporte a pagamento parcial

Contas a Pagar

  • Cadastre despesas da matriz
  • Categorize por tipo (Aluguel, Energia, Fornecedor, etc.)
  • Registre pagamentos realizados

8. Conciliacao Bancaria

Sistema estilo Conta Azul para controle bancario:

  1. Cadastre as contas bancarias da empresa
  2. Importe extratos (OFX, CSV, Excel)
  3. Use "Conciliar Automaticamente" para associar transacoes
  4. Concilie manualmente as pendentes

9. Notas Fiscais

O sistema suporta emissao de:

  • NF-e: Nota Fiscal Eletronica
  • NFC-e: Cupom Fiscal Eletronico
  • NFS-e: Nota Fiscal de Servico

Configure os dados fiscais antes de emitir notas.

10. Logs de Auditoria

O sistema registra todas as acoes importantes:

  • Quem criou/alterou cada registro
  • Data e hora da acao
  • Dados antes e depois da alteracao

11. Boas Praticas

  • Mantenha o estoque central sempre atualizado
  • Processe as solicitacoes das franquias rapidamente
  • Monitore o fluxo de caixa regularmente
  • Faca a conciliacao bancaria semanalmente
  • Revise os logs de auditoria periodicamente
  • Treine os gestores das franquias no uso do sistema
